PyGetMyPublicIP
============
This is a program that get and show your public IP addresses.  

## Features
* Support many websites:
    + https://myip.com.tw
    + https://www.rus.net.tw/myip.php
    + https://api.ipify.org
    + https://whatismyipaddress.com

## How to Use

```
getpublicip.py
```

## System Requirement
Python
